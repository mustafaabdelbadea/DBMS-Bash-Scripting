# DBMS-ShellScript

## Installation

### Platform & tools
You should use any linux distribution (redhat, ubuntu,etc..) 
### Get the Code

Either clone this repository or fork it on GitHub and clone your fork:

```
git clone https://github.com/mustafaabdelbadea/DBMS-ShellScript.git
cd DBMS-ShellScript
```

## Running
* Run the application

    ```
    chmod u+x ./dbms.sh
    chmod u+x ./colors.sh
    ./dbms.sh
    ```
